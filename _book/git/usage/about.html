1. 实战 git

git 是一款分布式版本控制系统,可以简单概括: 不要把鸡蛋放在一个篮子里,你的一举一动都在监视中.

1.1. 实战场景

你作为某项目的其中一员或者负责人,和小伙伴们一起开发,大家既有着各自分工互不干扰,也有着相互合作,最终每个人的劳动成果汇聚成最后的项目,愉快完成项目!

1.2. 要求

  • 理解 git 的工作流程,懂得实际工作中如何交流合作
  • 掌握 git 常用操作,工具为我所有,进而提高工作效率
  • 独当一面,最好能够独自解决使用git 过程中遇到的问题
  • 主动分享经验,能够教会别人如何使用 git 更上一层楼

1.3. 推荐

  • 最好的教程在官网 git 官网
  • 在线练习常用操作 Learning Git Branching
  • 廖雪峰的官方网站 git教程
